## Backend Project - Authentication and Authorization üîê

This repository contains a backend project that implements authentication and authorization using NestJS. It features Role-Based Access Control (RBAC) and integrates Cosmos DB for secure and scalable data storage. The entire system follows best practices for secure and efficient development.

## ‚öôÔ∏è Technologies Used

- NestJS: Backend framework for building a robust API structure.
- Cosmos DB: NoSQL database for scalable and secure user data storage.
- RBAC (Role-Based Access Control): Define user roles and permissions dynamically.
- JWT Authentication: Manage secure sessions with JSON Web Tokens.
- Password Encryption: Securely store credentials with encryption algorithms.

## üìå Features

1. Authentication:
- Login system with JWT tokens.
- Password recovery via email or WhatsApp OTP.

2. Authorization:
- Dynamic roles assignment using RBAC.
- Permission validation to control access to protected resources.

3. User Management:
- Complete CRUD for users and admins.
- Password strength validation (minimum 8 characters, symbols, numbers, etc.).

4. Cosmos DB Integration:
- Connection managed through a global module using ConfigService in NestJS.
- CRUD operations with dynamic data models.
